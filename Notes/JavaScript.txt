JavaScript is an interpreted language

Each browser has its javascript interpreter:
1) V8 : Chrome and Node.js
2) SpiderMonkey : FireFox
3) JavaScriptCore : Safari
4) Chakra : Microsoft Edge/ IE


Types : JavaScript has dynamic typing
Primitive Types:(no methods, immutable)
1) undefined
2) null
3) boolean
4) number
5) string
6) (symbol)

Typecasting : Coercion
Explicit vs Implicit coercion

const x = 42
const explicit = String(42)			// explicit === "42"
const implicit = 42 + ""			// implicit === "42"

== vs ===
== coerces types
=== requires equivalent types
